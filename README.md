## Feature

- Format support:
  - EPUB (**.epub**)
  - PDF (**.pdf**)
  - DRM-free Mobipocket (**.mobi**) and Kindle (**.azw3**, **.azw**)
  - FictionBook (**.fb2**)
  - Platform support: **Windows**, **macOS**, **Linux** and **Web**
  - Built-in 32 themes
  - Add notes, highlights to your books
